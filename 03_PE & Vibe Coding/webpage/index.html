엄광호 · 프롬프트 엔지니어링과 바이브 코딩

이 세미나는 무엇인가요?

프롬프트 엔지니어링(PE)으로 AI에게 잘 지시하는 법부터, 바이브 코딩으로 AI와 함께 직접 만들고 세상에 내놓기까지 — 실무에서 바로 쓰는 209장의 교안을 쉬운 해설로 풀었습니다.

추천 읽기 순서: 🧭 학습 흐름으로 큰 그림을 잡고 → 슬라이드 뷰로 하나씩 따라간 뒤 → 모르는 용어는 📚 개념 사전에서 확인하세요.

📚 개념 사전

      
세미나에 등장한 핵심 개념을 웹 검색으로 검증해 쉽게 풀었습니다. 왼쪽에서 개념을 고르면 오른쪽에 상세 해설이 열립니다. (모델 지식) 표시는 검증 출처 없이 일반 지식으로 보충한 부분입니다.

    

    

      

      

    

  


  
  

    

      
⭐ 강사가 강조한 내용

      
발표에서 특별히 강조·당부하거나, 실무 조언·주의점·평가로 짚은 지점만 모았습니다. 카테고리 버튼으로 걸러 볼 수 있고, 각 항목의 슬라이드 링크로 이동합니다.

    

    

    

  


  
  

    

      
🧭 학습의 흐름

      

      
갈래(track)를 고르고 단계를 클릭하면 개념 해설이 열립니다. 단계 사이 화살표는 "이전 단계 → 다음 단계로 넘어가는 이유"를 보여줍니다. 다음 개념 ▶으로 흐름을 한 단계씩 따라갈 수 있습니다.

    

    

    

  


  
  

    

      
    

    
대소문자 구분 없이 검색됩니다. 여러 단어는 모두 포함(AND)한 슬라이드가 우선 노출됩니다.
